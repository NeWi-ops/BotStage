# Offre de Stage : Stagiaire Data Engineer / MLOps @ Carrefour AI Lab

**Lieu :** Massy (91) / Télétravail partiel possible
**Durée :** 6 mois
**Début :** Dès que possible

## Description du Poste
Au sein du AI Lab de Carrefour, vous rejoindrez l'équipe "Plateforme" chargée de mettre en production les modèles de Machine Learning (Recommandation, Prédiction de stocks, Pricing). Votre rôle sera de construire des pipelines de données robustes et d'industrialiser les flux.

## Missions
- Concevoir et déployer des pipelines ETL/ELT asynchrones en Python.
- Participer à la mise en production de modèles (MLOps) via Docker et Kubernetes.
- Automatiser les tests et le déploiement (CI/CD) sur la Google Cloud Platform (GCP).
- Optimiser les requêtes BigQuery pour le traitement de gros volumes de données.

## Profil Recherché (Compétences Clés)
- Étudiant(e) en dernière année d'école d'ingénieur ou Master spécialisé en IA/Data.
- **Maitrise indispensable : Python (FastAPI/Flask), SQL.**
- Expérience pratique avec : Docker, Git (GitLab CI).
- Connaissance de GCP (BigQuery, Dataflow) ou AWS est un gros plus.
- Curiosité pour le MLOps et l'automatisation.
