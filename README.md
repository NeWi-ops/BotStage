# 🤖 Sourceur de Stages - Telegram Bot

Ce projet est un bot Telegram intelligent conçu pour lire un flux RSS d'offres de stage et trouver les meilleures correspondances avec le CV de l'utilisateur grâce à l'IA (LLM via OpenRouter).

## 🛠️ Installation

1. Cloner le repository.
2. Installer les dépendances :
   ```bash
   pip install python-telegram-bot openai feedparser requests python-dotenv
